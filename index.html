Sekilas Info

Pragmatic UI dibikin untuk kamu yang masih belajar membuat aplikasi Website, atau lagi ada Project yang butuh tampilan keren tapi mudah dibuat dengan menggunakan Framework CSS Boostrap.
Berikut hal yang perlu kamu ketahui:

  • Bagi pemula, pelajari Boostrap basic seperti Grid, Navbar, dan beberapa hal yang lain
  • Masih belum support custom color pada component, jadi kamu harus melakukan penambahan style untuk merubah color
  • Ayo kontribusi untuk mengembangkan Pragmatic UI yang lebih baik!
